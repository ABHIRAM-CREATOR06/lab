# University Lab Assignments

Welcome to my central repository for all collegiate laboratory coursework and practical assignments. This repository is structured to organize the various assignments and projects completed across different computer science modules.

## Repository Structure

The repository is divided into several main directories, each dedicated to a specific subject:

* **[Java Lab](./Java%20Lab/)**: Contains practical exercises and project files focusing on Object-Oriented Programming principles using Java.
* **[OS Lab](./OS%20Lab/)**: Includes implementations of Operating System concepts, algorithms (such as CPU scheduling, page replacement), and system administration scripts.
* **[ds lab](./ds%20lab/)**: Dedicated to Data Structures and Algorithms. Features implementations and problem-solving exercises for arrays, linked lists, stacks, queues, trees, and graphs.
* **[network lab](./network%20lab/)**: Contains networking assignments, including socket programming, network topology simulation, and protocol implementations.

## How to Navigate

Each folder contains the respective source code and potentially specific instructions or problem statements for those labs. Feel free to explore the directories for detailed implementations.
